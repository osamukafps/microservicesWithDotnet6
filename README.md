# ***Microservices Using .NET 6***
## Estudando Arquitetura de Microsserviços utilizando o .NET 6.
_Neste projeto serão utilizadas as seguintes tecnologias:_

* **.NET 6**
* **Oauth2**
* **OpenID**
* **JWT (JSON Web Tokens)**
* **Identity Server**
* **RabbitMQ**
* **API Gateway com Ocelot**
* **Swagger OpenAPI**
